<h2>TechHire</h2>

Laravel SQLite

TechHire is a dedicated recruitment platform connecting top-tier tech talent with innovative companies. Unlike generic job boards, this application is tailored specifically for the software/tech industry, featuring specialized filtering for programming languages, frameworks, and remote capabilities.

View Demo!

Advanced Job Search & Filtering

Implemented a dynamic search engine allowing users to query opportunities by keywords and filter results based on specific tech stacks, job types, and experience levels.

Comprehensive Job Insights

Designed detailed job views that present candidates with essential role information, including salary ranges, location requirements, and technical skill requirements for precise matching.

Employer Recruitment Hub

Built a secure dashboard for employers to manage job postings (CRUD) and review candidates. Includes functionality to download resumes and update application status.

Applicant Tracking System

A personalized user dashboard allowing job seekers to track their application history, monitor real-time status updates, and manage their professional profiles and portfolios.
