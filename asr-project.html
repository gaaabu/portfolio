<h2>ASR-induced Ambiguity Correction</h2>

Whisper mT5 MER M² F-Score

A hybrid post-processing pipeline designed to correct ASR-induced lexical ambiguities in Filipino-English (Taglish) code-switched speech. Utilizes rule-based error simulation to train a mT5 model to fix ASR errors without retraining the core system.

WHISPER + mT5

Hybrid Pipeline Architecture

We employ a Rule-Based Error simulation to generate synthetic "noisy-clean pairs", mimicking common Taglish ASR errors (e.g., homophones). These pairs fine-tune the mT5-base transformer.

50,000 AUGMENTED PAIRS

Dataset & Performance

Trained on a curated corpus of 50,000 sentence pairs derived from YouTube audio. The fine-tuned model achieved a Mixed Error Rate(MER) of 3.56, down from a 37.55 baseline.
